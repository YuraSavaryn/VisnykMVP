# Visnyk

Visnyk is a news collection and analysis system that combines 
classical machine learning methods with modern LLM technologies.

Key Features:
- Automated Data Pipeline: Parsing of raw news, data processing, and storage in the database.
- News Clustering: Grouping news by topics.
- News Ranking: Prioritizing news by importance.
- AI Assistant: An agent with access to RAG and web search tools.

Tech Stack:
- Backend: Python(FastApi, Celery)
- Frontend: React
- Database: PostgreSQL + pgvector
- Message Broker: Redis
- ML Frameworks: scikit-learn, langchain, transformers
